# San Jacinto College Chatbot

Led development of a bilingual (English/Spanish) AI-powered chatbot for San Jacinto College's website to assist current and prospective students with information and guidance.

## Role

Technical Lead (February - March 2022)

## The Challenge

San Jacinto College needed an intelligent chatbot to help students navigate college resources, answer frequently asked questions, and provide assistance in both English and Spanish to serve their diverse student population. The project had an aggressive 8-week timeline.

## Key Achievements

- **Complete delivery** within 8-week timeline
- **Bilingual support** with seamless English/Spanish switching
- **Customizable frontend** leveraging Watson Assistant features
- **Successful production deployment**
- **Positive client feedback**

## Technical Implementation

### Watson Assistant Integration

Built a custom React frontend that integrates with IBM Watson Assistant for natural language processing. The system handles complex dialogue flows and provides contextual responses based on student queries.

### Bilingual Architecture

Implemented seamless language switching between English and Spanish, ensuring the chatbot could serve all students effectively regardless of their preferred language.

### Containerized Deployment

Created a containerized solution using Docker and deployed on Kubernetes for scalability and reliability. The architecture allows the college to easily scale the chatbot based on usage patterns.

## Technologies Used

- React with JavaScript
- IBM Watson Assistant
- Node.js backend
- Docker containerization
- Kubernetes orchestration

## Leadership & Process

- Led agile development team through 8-week sprint cycles
- Conducted daily standups and sprint planning
- Coordinated with stakeholders on requirements
- Ensured rapid iteration and delivery

## Impact

The chatbot now assists students on the San Jacinto College website, providing instant answers to common questions and helping students navigate college resources. The bilingual capability ensures accessibility for the college's diverse student population.
