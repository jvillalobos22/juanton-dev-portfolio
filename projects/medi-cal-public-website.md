# Medi-Cal Public Website

Led Squad 3 in the modernization of California's Medi-Cal public website, transforming the legacy site into a modern React application with strict accessibility compliance and responsive design.

## Role

Development Lead - Squad 3 (October 2022 - July 2023)

## The Challenge

California's Medi-Cal public website needed a complete overhaul to improve accessibility, user experience, and maintainability. The project required migrating 1000+ pages from a legacy system while establishing accessibility compliance standards and creating a scalable design system.

## Key Achievements

- **500+ story points** delivered over 10 months
- **Team scaling** from 2 to 7 developers
- **Atomic design style guide** created from scratch
- **1000+ pages** migrated from legacy system
- **Accessibility compliance** standards established

## Technical Implementation

### Atomic Design System

Architected and implemented an atomic design system that provided consistent UI patterns across the entire website. Created a hierarchy of components from atoms to pages, enabling rapid development and maintaining design consistency.

### Content Management

Integrated Directus CMS for content management, enabling non-technical stakeholders to update website content without developer intervention while maintaining design integrity.

### Accessibility Standards

Implemented WCAG 2.1 compliance standards throughout the application, including keyboard navigation, screen reader support, color contrast requirements, and semantic HTML.

## Technologies Used

- React with TypeScript
- GraphQL
- Directus CMS
- OpenShift
- Adobe XD
- ServiceNow

## Leadership & Process

- Ran all sprint ceremonies (planning, standups, retrospectives)
- Facilitated design thinking workshops
- Created design and development workflows
- Collaborated directly with DHCS stakeholders
- Mentored team members through pair programming

## Impact

Successfully modernized California's Medi-Cal public website, improving accessibility for millions of California residents seeking healthcare information. The atomic design system established patterns that accelerated development and ensured consistency across the entire site.
